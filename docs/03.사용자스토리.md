### US-001: 회원가입

**스토리**

> 블로그 방문자로서,
> 나는 이메일로 회원가입하고 싶다.
> 왜냐하면 나만의 글을 작성하고 관리하고 싶기 때문이다.

**인수 조건**

- [ ] 이메일과 비밀번호 입력 폼이 있다
- [ ] 비밀번호는 6자 이상이어야 한다
- [ ] 가입 완료 시 자동으로 로그인된다
- [ ] 이미 가입된 이메일로는 가입할 수 없다

---

### US-002: 소셜 로그인

**스토리**

> 바쁜 사용자로서,
> 나는 Google 계정으로 빠르게 로그인하고 싶다.
> 왜냐하면 새 비밀번호를 만들고 기억하기 귀찮기 때문이다.

**인수 조건**

- [ ] Google 로그인 버튼이 있다
- [ ] 클릭 시 Google 로그인 팝업이 뜬다
- [ ] 로그인 성공 시 메인 페이지로 이동한다
- [ ] 처음 로그인해도 별도 회원가입 절차가 없다

---

### US-003: 게시글 작성

**스토리**

> 로그인한 사용자로서,
> 나는 새로운 블로그 글을 작성하고 싶다.
> 왜냐하면 오늘 배운 내용을 기록하고 공유하고 싶기 때문이다.

**인수 조건**

- [ ] 제목과 내용을 입력할 수 있다
- [ ] 카테고리를 선택할 수 있다 (선택사항)
- [ ] 발행 버튼을 누르면 글이 저장된다
- [ ] 저장 후 작성한 글 페이지로 이동한다

---

### US-004: 게시글 목록 보기

**스토리**

> 블로그 방문자로서,
> 나는 최신 글 목록을 보고 싶다.
> 왜냐하면 어떤 글이 있는지 한눈에 파악하고 싶기 때문이다.

**인수 조건**

- [ ] 로그인하지 않아도 글 목록을 볼 수 있다
- [ ] 글은 최신순으로 정렬된다
- [ ] 각 글의 제목, 작성자, 날짜가 표시된다
- [ ] 글 제목을 클릭하면 상세 페이지로 이동한다

---

### US-005: 내 글 수정

**스토리**

> 글 작성자로서,
> 나는 이미 발행한 글을 수정하고 싶다.
> 왜냐하면 오타를 발견했거나 내용을 보완하고 싶기 때문이다.

**인수 조건**

- [ ] 내가 쓴 글에만 수정 버튼이 보인다
- [ ] 수정 페이지에서 제목과 내용을 변경할 수 있다
- [ ] 저장 시 수정된 내용이 반영된다
- [ ] 다른 사람의 글은 수정할 수 없다

---

### US-006: 내 글 삭제

**스토리**

> 글 작성자로서,
> 나는 더 이상 필요 없는 글을 삭제하고 싶다.
> 왜냐하면 블로그를 정리하고 싶기 때문이다.

**인수 조건**

- [ ] 내가 쓴 글에만 삭제 버튼이 보인다
- [ ] 삭제 전 확인 메시지가 표시된다
- [ ] 삭제 후 목록에서 해당 글이 사라진다
- [ ] 다른 사람의 글은 삭제할 수 없다

---

### US-007: 반응형 화면

**스토리**

> 모바일 사용자로서,
> 나는 스마트폰에서도 블로그를 편하게 이용하고 싶다.
> 왜냐하면 이동 중에도 글을 읽고 싶기 때문이다.

**인수 조건**

- [ ] 모바일 화면에서 레이아웃이 깨지지 않는다
- [ ] 버튼과 링크가 터치하기 적절한 크기다
- [ ] 텍스트가 읽기 편한 크기다

---

### US-008: 댓글 작성

**스토리**

> 로그인한 사용자로서,
> 나는 게시글에 댓글을 작성하고 싶다.
> 왜냐하면 글에 대한 의견이나 질문을 남기고 싶기 때문이다.

**인수 조건**

- [ ] 게시글 상세 페이지에 댓글 입력란이 있다
- [ ] 로그인한 사용자만 댓글을 작성할 수 있다
- [ ] 비로그인 상태에서 댓글 작성 시 로그인 페이지로 이동한다
- [ ] 댓글 작성 후 목록에 바로 표시된다

---

### US-009: 댓글 삭제

**스토리**

> 댓글 작성자로서,
> 나는 내가 작성한 댓글을 삭제하고 싶다.
> 왜냐하면 잘못 작성했거나 더 이상 필요 없는 댓글을 정리하고 싶기 때문이다.

**인수 조건**

- [ ] 내가 쓴 댓글에만 삭제 버튼이 보인다
- [ ] 삭제 전 확인 메시지가 표시된다
- [ ] 삭제 후 목록에서 해당 댓글이 사라진다
- [ ] 다른 사람의 댓글은 삭제할 수 없다

---

### US-010: 댓글 목록 보기

**스토리**

> 블로그 방문자로서,
> 나는 게시글의 댓글 목록을 보고 싶다.
> 왜냐하면 다른 사람들의 의견과 토론을 확인하고 싶기 때문이다.

**인수 조건**

- [ ] 로그인하지 않아도 댓글 목록을 볼 수 있다
- [ ] 각 댓글의 작성자, 내용, 작성일이 표시된다
- [ ] 댓글은 작성 순서대로 정렬된다
